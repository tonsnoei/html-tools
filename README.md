# Browser-Based HTML Tools

A collection of fully client-side web tools that run entirely in your browser. No backend servers, no data uploaded to the cloud — everything stays local on your machine.

---

## Categories

### File Comparison & Management

- **[Compare Files](https://tonsnoei.github.io/compare-files/)** — Side-by-side file comparison tool. Diff text files, spot changes, and merge differences entirely in the browser. No uploads, no servers.

- **[HTML Link Manager](https://tonsnoei.github.io/html-link-manager/)** — A local link dashboard for managing and organizing bookmarks. Save, categorize, and search your links with everything stored in-browser.

### Security & Encryption

- **[Secure WeTransfer](https://tonsnoei.github.io/secure-wt/)** — Encrypt files client-side before uploading to WeTransfer or any other file-sharing service. Your data is encrypted in the browser so only someone with the passphrase can decrypt it. Also works for general-purpose file encryption.

### PDF Tools

- **[PrivatePDF](https://privatepdf.eu/)** — Organize and manipulate PDFs locally. Merge, split, reorder, and manage pages without sending your documents to any server.

### AI & LLM Tools

- **[Qwen3.6 VRAM Calculator](https://tonsnoei.github.io/qwen36-vram-calculator/)** — Estimate VRAM requirements for running Qwen3.6 models at various quantization levels and context lengths. Helps you determine which model variant fits your GPU before downloading.

---

All tools are static sites. Nothing leaves your device unless you explicitly upload or share it yourself.
